# 🌳 Urban Forest Survival Intelligence System

An AI-powered platform for tracking, monitoring, and managing urban tree plantation initiatives from planting to maturity.

## 🎯 Features

### Core Features
- **Tree Registration**: Digital registration with geo-tagging
- **Role-Based Access**: Admin and Visitor login modes
- **Care Logging**: Track watering, fertilizing, pruning activities
- **Photo Timeline**: Upload and track tree growth photos
- **Public Dashboard**: Transparent view of all planted trees
- **Survival Tracking**: Monitor tree health and survival status

### AI/ML Features
- **Image-Based Health Analysis**: Upload photos for ML health assessment
- **Survival Prediction**: ML-driven survival probability scoring
- **Care Recommendations**: Automated, intelligent care suggestions
- **Species Templates**: Pre-configured care schedules by species
- **Alert System**: Proactive notifications for overdue care

### Impact Metrics
- Carbon sequestration tracking (CO₂ captured)
- Oxygen output measurement
- Survival rate analytics
- Maintenance compliance tracking

## 🚀 Quick Start

### Installation

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

## 📱 Usage

1. **Login**: Choose Admin or Visitor role
2. **Dashboard**: View overall statistics and tree locations
3. **Tree Registry**: Browse all registered trees
4. **Tree Details**: View individual tree health, care history, and AI recommendations
5. **Care Log**: Add maintenance activities (Admin only)
6. **Register Tree**: Add new trees to the system (Admin only)

## 🏗️ Tech Stack

- **Frontend**: React 18 + Vite
- **Routing**: React Router v6
- **Maps**: Leaflet + React-Leaflet
- **Charts**: Recharts
- **Icons**: Lucide React
- **Styling**: Custom CSS
